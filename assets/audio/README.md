# Audio Files

This directory contains audio files for guided sessions.

## Required Files

### breathing-guide.mp3
- Duration: ~3 minutes
- Type: Guided breathing meditation
- Description: A calm, soothing voice guiding the user through breathing exercises
- Tone: Gentle, supportive, grounding

## Audio Guidelines

- Use royalty-free or licensed audio
- Keep volume levels consistent
- Ensure clear, high-quality voice recording
- Consider adding subtle ambient background sounds (ocean waves, gentle rain)
- File format: MP3 (recommended) or M4A
- Bitrate: 128kbps or higher
